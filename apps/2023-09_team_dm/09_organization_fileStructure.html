Organization - Directory Structure & File Naming

Lifecycle elements: obtain/collect, store/backup, find, share w. team, use, document, preserve, share outside of team

  • Define folder and file names and structure - and use it
  • Logically separate raw/source data from derived data and products
  • Use meaningful names that include basic information (e.g. date, measurement, collection, etc.)
  • Unique
  • Avoid Spaces
  • ASCII Characters only
